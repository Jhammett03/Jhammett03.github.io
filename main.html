Currently Building

Attention Tracker

Senior project — a computer vision pipeline that classifies focused vs. distracted states from 10-second video clips. MediaPipe extracts 478 facial landmarks per frame at 3 fps; from these I engineer per-frame Eye Aspect Ratio (EAR) and eye-center coordinates, then compare four architectures on the same dataset using 5-fold stratified cross-validation.

Video (10s) → MediaPipe (478 landmarks) → EAR + gaze features → LR / 1D CNN / Transformer → Focused / Distracted

Applications: driver drowsiness detection, student engagement analysis, HCI research.
